# Student Management Information System (SMIS) Requirements

## Project Objective
To design and implement a backend system managing academic records, focusing on data integrity and relational mapping.

## Functional Requirements
* Record academic departments and physical locations.
* Maintain student profiles with unique identification.
* Catalog courses with credit-based validation.
* Track enrollments, semesters, and grading history.

## Technical Constraints
* Database: Microsoft SQL Server 2025.
* Normalization: 3rd Normal Form (3NF).
* Primary Keys: Auto-incrementing integers (Identity).
